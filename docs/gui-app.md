# GUIアプリの現状

## 対象店舗

- 現在は MJアリーナ箱崎店 固定です
- 店舗URLは画面に表示しますが、変更はしない前提です
- `登録店舗` タブでは、あとで使うための店舗URLを入力して店舗名を自動取得し、仮登録できます

## 取得の流れ

1. 対象日を入力する
2. 機種一覧を読み込む
3. 表として表示された機種一覧の左端にあるチェックボックスを押して取得したい機種を選ぶ
4. 必要なら各列の見出しを押して昇順と降順を切り替える
5. 取得ボタンを押す

## 機種一覧について

- 店舗ページから対象日のページを開き、その日の機種一覧を読み込みます
- 複数台設置の機種と、一台設置の機種をどちらも一覧に含めます
- 機種一覧は表で表示し、左端にチェックボックス、続いて機種名、台数、平均差枚、平均G数、勝率、出率を並べます
- 機種一覧を読み込んだ直後の表示順は、台数の多い順です
- 機種一覧と台データは、各列の見出しを押すたびに昇順と降順を切り替えられます
- 店舗によって設置機種が異なるため、対象日ごとに一覧を読み込み直す想定です

## 結果表示

- 選択した機種の台データをまとめて表に表示します
- 機種によって表示される項目数が違う場合は、足りない項目を空欄で表示します

## 登録店舗タブ

- 店舗URLを入力すると、ページから店舗名を自動取得してアプリ内の一覧へ仮登録できます
- 同じ店舗名または同じURLは重複登録しないようにしています
- 今は見た目と流れを作った段階で、外部への保存はしていません
