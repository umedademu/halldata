# GUIアプリの現状

## 対象店舗

- データ取得タブでは、登録店舗の一覧から対象店舗を選びます
- 初期状態では MJアリーナ箱崎店 を選んでいます
- 店舗URLは選んだ店舗に応じて表示が切り替わります
- `登録店舗` タブでは、店舗URLを入力するとページから店舗名を自動取得して登録できます
- 登録した店舗一覧は `local_data` と `Supabase` に保存され、次回起動時はローカル保存分を読み込みます
- 店舗URLは保存前に表記ゆれをそろえるため、同じURLの大文字小文字違いなどで重複登録されにくくしています

## 取得の流れ

1. 登録店舗の中から対象店舗を選ぶ
2. 対象期間を `YYYY-MM-DD ～ YYYY-MM-DD` の形で入力する
3. 機種一覧を読み込む
4. 表として表示された機種一覧の左端にあるチェックボックスで取得したい機種を選ぶ
5. 必要なら `取得後に台データ表を表示しない` にチェックを入れる
6. 取得ボタンを押す
7. 取得前に、同じ店舗、同じ日付、同じ機種の保存済みデータが `local_data` と `Supabase` にあるかを確認し、見つかった分はスキップする
8. 取得中は、状態欄の下にある進捗バーで日付ごとの確認と機種ごとの取得の進み具合を見られる
9. 各日付の取得が終わるたびに、その日分だけローカル保存と `Supabase` 保存を自動で行う
10. 最後に、日付ごとの取得結果をまとめて画面へ表示する

## 機種一覧について

- 店舗ページから期間の終了日にあたる日付ページを開き、その日の機種一覧を読み込みます
- 複数台設置の機種と、一台設置の機種をどちらも一覧に含めます
- 機種一覧は表で表示し、左端にチェックボックス、続いて機種名、台数、平均差枚、平均G数、勝率、出率を並べます
- 機種一覧を読み込んだ直後の表示順は、台数の多い順です
- 機種一覧の各列は、見出しを押すたびに昇順と降順を切り替えられます
- 店舗によって設置機種が異なるため、対象期間を変えたら機種一覧も読み込み直す想定です

## 台データ比較について

- 取得結果は、1列目を日付列として表示します
- `台データ表を広く表示` ボタンを押すと、他の項目をたたんで台データ表だけを大きく表示できます
- `日付末尾` のプルダウンで `0` から `9` を選ぶと、その数字で終わる日だけに絞って表示できます
- 横にスクロールしても、左端の日付列は固定したまま表示します
- 日付列は右側の表と横の余白なしで続けて表示します
- 日付列の行を押すと、その日付の横一列を強調表示できます
- 取得中は、状態欄の下にある進捗バーで全体の進み具合を確認できます
- `取得後に台データ表を表示しない` を使うと、取得結果の保存は行いつつ、重い比較表の描画だけを省略できます
- 後でチェックを外すと、直前に取得した結果から台データ表を表示できます
- 再描画後も、表の表示位置と上下のスクロールバーがずれにくいように調整しています
- すでに保存済みの同一日付、同一機種のデータは取得時にスキップし、概要欄にスキップ件数を出します
- 1行目の見出しには台番を並べます
- 2行目の見出しには、その台ごとの `機種名` `差枚` `G数` `出率` `BB` `RB` `合成` `BB率` `RB率` を並べます
- 行は日付ごとに並ぶため、同じ台の流れを日をまたいで見比べられます
- たとえば `7` を選ぶと、`3/7` `3/17` `3/27` `4/7` のような日だけを並べられます
- 台データ比較の各子見出しも、押すたびに昇順と降順を切り替えられます
- 表示が重くなりにくいよう、台データ比較は軽い描き方に見直しています
- 指定期間の中で機種が見つからない日は、その日の欄を空欄のまま残します

## 自動保存について

- 取得中は、各日付の取得が終わるたびに、その日分を `local_data` 配下へ `json` で自動保存します
- あわせて、その日分を `env.local` の設定を使って `Supabase` の `stores` と `machine_daily_results` に自動保存します
- 次回以降の取得では、このローカル保存分と `Supabase` 保存分の両方を見て、取得済みの日付と機種を飛ばします
- `env.local` では最低限 `SUPABASE_URL` と `SUPABASE_SERVICE_ROLE_KEY` を使います
- `env.local` が未設定、または `Supabase` 保存に失敗した場合でも、取得済みの表表示は残します
- `env.local` と `local_data` は追跡対象から外す想定です

## 登録店舗タブ

- 店舗URLを入力すると、ページから店舗名を自動取得して一覧へ登録できます
- 同じ店舗名または同じURLは重複登録しないようにしています
- 登録後は `local_data` に保存し、あわせて `Supabase` の `stores` にも保存します
- 次回起動時は、ローカル保存済みの登録店舗一覧を読み込んで表示します
