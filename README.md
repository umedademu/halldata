パチスロ店のデータを収集するアプリ

現在の叩き台版では、`apps/gui` の画面から MJアリーナ箱崎店 の対象日データを取得できます。
先に登録店舗から対象店舗を選び、対象期間を入力し、期間の終了日を基準に機種一覧を読み込み、チェックした機種をまとめて取得する流れです。
また、`登録店舗` タブでURLから店舗名を自動取得して登録でき、登録店舗一覧は `local_data` と `Supabase` に保存します。
取得結果は、1列目を日付、その右に台番ごとの比較列を並べる形で表示します。
取得完了後は、`local_data` へのローカル保存と `Supabase` への保存を自動で行います。
`Supabase` 保存には、`env.local` の `SUPABASE_URL` と `SUPABASE_SERVICE_ROLE_KEY` を使います。
`apps/web` には、`Supabase` に保存された内容を `店舗一覧 → 機種一覧 → 台データ` の順で見られるサイトを用意しています。
このサイトは `PC` と `スマホ` の両方で見やすいように、広い画面では比較表、狭い画面では日付ごとのカード表示に切り替わります。
